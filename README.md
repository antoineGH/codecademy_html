## Setup

To run this project, clone it and start index.html

```
$ git clone https://github.com/antoineratat/codeacademy_bootstrap.git
$ cd codeacademy_bootstrap
```

For templars site

```
$ cd templars
```

For adhoc site

```
$ cd adhoc
```

For checkerboard

```
$ cd checkerboard
```

For bootstrap grid

```
$ cd test
```

For paint site

```
$ cd project_bootstrap_paint
```
